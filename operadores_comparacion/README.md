# OPERADORES DE COMPARACION
Los operadores de comparacion se utiliza para comparar 2 valores y devuelven un valor booleano `(True o False)` segun si la comparacion es verdadeera o falsa.
# Igualdad(==)
Compara si dos valores son iguales.
>ejemplo
```python
a=5
b=3
resultado= a == b
print(resultado) 
#imprime False
```
# Desigualdad(!=)
Compara si dos valores son diferentes:
>ejemplo
```python
a=10
b=5
resultado=a != b
print(resultado) 
 # imprime True
```
# Mayor que (>)
compara si un valor es mayor que el otro.
> ejemplo
```python
a=20
b=12
resulatado=a > b
print(resulatdo)  
 #imprime True
```
# Menor que (<)
Compara si un valor es menor que otro.
>ejemplo
```python
a=21
b=18
resultado=a < b
print(resultado) 
 #imprime False
```
# Mayor o Igual que (>=)
Compara si un valor es mayor que otro.
>ejemplo
```python

b=20
resultado=a >= b
print(resultado) 
 #imprime False
```
